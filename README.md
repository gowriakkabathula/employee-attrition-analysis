# HR Analytics: Employee Attrition Analysis
# Project Overview
This project analyzes employee attrition data to identify the major factors affecting employee retention in an organization. The analysis focuses on understanding patterns related to age groups, salary, overtime, promotions, work-life balance, and job satisfaction.
# Objectives
1. Analyze employee attrition patterns
2. Identify high-risk employee groups
3. Understand factors affecting employee retention
4. Create meaningful visualizations
5. Generate business recommendations using data insights
# Tools & Technologies Used

4. Matplotlib
5. Seaboarn
6. Jupyter Notebook
# Dataset Used:
IBM HR Analytics Employee Attrition Dataset

# Key Insights
1. Employees aged 26-35 show higher attrition rates.
2. Employees working overtime are more likely to leave the company.
3. Lower monthly income appears to be associated with higher attrition.
4. Some departments experience higher attrition compared to others.
5. Employees with poor work-life balance tend to leave more frequently.
6. Delayed promotions may contribute to employee attrition.
# Business Recommendations
1. Provide competitive salary hikes for lower-income employees
2. Improve promotion oppurtunities for long-serving employees
3. Reduce excessive overtime workload
4. Improve work-life balance initiatives
5. Conduct employee engagement and feedback programs
6. Focus retention strategies on high attrition age groups
